## What is BrandForge AI?

BrandForge AI is an end-to-end multi-agent system that transforms any company
website URL into a complete, professionally-designed brand kit. It demonstrates
practical application of agentic AI, retrieval-augmented generation (RAG), and
real-time streaming UI in a production-grade system вҖ” not a notebook demo.

**The flow:** scrape вҶ’ understand вҶ’ generate вҶ’ review вҶ’ package, with 4 creative
agents running in parallel and a critic agent validating every output before
assets are rendered.

## Tech Stack

| Layer | Technology | Why |
|---|---|---|
| Orchestration | LangGraph | Explicit state machine with parallel fan-out/fan-in вҖ” not a black-box agent wrapper |
| LLM | Groq (Llama 3.3 70B) + Gemini fallback | Fast inference, automatic failover on rate limits |
| RAG | ChromaDB + sentence-transformers | Local, zero-cost vector search grounding every agent in real brand content |
| Scraping | Jina Reader API + Playwright | Handles both static and JS-rendered sites with real link discovery |
| Backend | FastAPI + SSE | Async job queue with live progress streaming |
| Asset rendering | Playwright (HTMLвҶ’PDF/PNG) | Pixel-perfect designs using real CSS вҖ” not programmatic canvas drawing |
| Frontend | Next.js 14, TypeScript, Tailwind | App Router, Server Components, fully typed API contract |

## Key Technical Decisions

*   **Parallel agent execution** вҖ” Copywriter, Layout, Email, and Ad agents run simultaneously via LangGraph's Send() API, then fan-in to a Critic agent. This isn't sequential prompting; it's genuine concurrent multi-agent design.
*   **Deterministic color extraction** вҖ” Brand colors are parsed directly from CSS/HTML, never hallucinated by the LLM, with a contrast-guarantee system that works across monochrome, dark-primary, and pastel brand archetypes.
*   **HTML-to-PDF/PNG rendering** вҖ” Assets are designed in HTML/CSS and rendered via Playwright, the same engine as Chrome DevTools, giving real typography, gradients, and layout вҖ” not primitives drawn with a graphics library.
*   **Real-time streaming architecture** вҖ” SSE events carry structured payloads (not just status strings), letting the frontend render a live brand profile card the moment extraction completes, before the pipeline finishes.

## Setup

### Prerequisites

*   Python 3.11+
*   Node.js 18+
*   Free API keys: Groq and Google AI Studio

### Backend

```bash
cd backend
pip install -r requirements.txt
playwright install chromium
cp .env.example .env
# Edit .env and add your GROQ_API_KEY and GEMINI_API_KEY
python main.py
```
Backend runs at `http://localhost:8000`

### Frontend

```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
```
Frontend runs at `http://localhost:3000`

### Try it

Open `http://localhost:3000`, paste any company URL (e.g. stripe.com), and watch the pipeline run live.
